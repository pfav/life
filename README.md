# A Curse(ed) Life

### Conway's Game of Life
Simple thing I like to play once in a while. 

### instructions

tested on Ubuntu Xenial 16.04.3 LTS

```bash
# dev deps
sudo apt-get install libncurses5 libncurses5-dev libncurses5-doc 

# build
make

# clean
make clean

# install (sudo)
make install 

```

### gameplay

* `c` clear world
* `f` fill world
* `r` reload world
* `p` pause
* `k` increment delay
* `j` decrement delay
* `MOUSE CLICK` toogle life at position
* `q` quit
